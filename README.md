# Nethernet

Nethernet is a high-performance 3D visualization engine that maps the history of the "dead internet", including the forums, MMOs, personal homepages, and hosting platforms that defined the early web but have since faded into the void.

## What is it?

The internet is often viewed as permanent, but vast swathes of human culture have been lost to link rot, server shutdowns, and corporate consolidation. Nethernet acts as a spatial time machine, allowing you to traverse the "sunken cities" of our digital past. Using real-time procedural rendering, it turns archival data from the Wayback Machine into an explorable, atmospheric 3D environment.

## The Experience

- **Spatial Navigation**: Glide through a dark, high-fidelity void where data clusters represent distinct digital eras.
- **Ghost Links**: Interact with "Ghost Links"—procedurally generated nodes that link directly to archived snapshots of abandoned websites.
- **Kinetic Data**: Watch the architecture of the internet shift and pulse based on temporal data and user interaction.

## Technical Architecture

Nethernet is built for performance and portability, leaning on modern web standards to provide a AAA visual experience without heavy external dependencies.

- **Rendering Engine**: Three.js/WebGL for high-performance 3D spatial environments.
- **Data Integration**: Dynamically fetches archival snapshots via the Internet Archive Wayback Machine API.
- **UI/UX**: Custom-built terminal-style interface using raw CSS and optimized browser APIs.
- **No Bloat**: Dependency-minimal architecture; optimized for sub-second load times.
